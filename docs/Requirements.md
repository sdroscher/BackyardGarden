# BackyardGarden

A web application that allows users to create and manage their own virtual gardens. The application provides a user-friendly interface for designing and customizing gardens, as well as tools for tracking plant growth and maintenance.

The main focus is for organizing planting schedules for seeds, but the app also supports tracking growth of existing plants.

## Required Features

- Seed database with planting schedules and care instructions. See Resources below for seed providers.
- Mobile-responsive design
- Seasonal planting guides and tips
- Customizable notifications for planting, watering, fertilizing, and harvesting
- Integration with local gardening resources (e.g. seed providers and nurseries)

## Optional Features

- Plant database with care instructions
- Plant growth tracking and reminders
- User authentication and profile management
- Integration with weather APIs for optimal planting schedules
- Data export and backup options
- Integration with smart gardening devices (e.g., soil moisture sensors, automated watering systems)
- Garden design and customization

## Initial data

A garden can be created in the app for simon@droscher.com with the garden data based on the file `Seed Planting 2026.csv` in the `data/` directory of the repo.

## Tech Stack

This can be anything appropriate for a web application, but ideally not node-based.
